<br/>
<br/>

## 개요


### - 하루에 하나씩 일기를 쓰고 그날의 감정을 기록하고, 후에 그 기억을 되새기며 회고할 수 있는 다이어리입니다.
### - 주기별 혹은 분기별로 감정을 차트로 보여주어 지난날의 감정을 돌아볼수있습니다.
<br/>
<br/>


## 기술언어

- 개발 환경 : visual code
- 기술 스택 : TypeScript, JavaScript, Tailwind, Sass, NextJS
- 외부 자원 : Radix UI, Gpt

<br/>
